Написать отчет, который показывает агрегированную сумму покупок по заказчику и материалу за определенный год,  
то есть вывести в отчете, какой заказчик какие материалы на какую сумму купил. 
 
Заголовки заказов находятся в таблице VBAK, позиции заказов в таблице VBAP,  
связь между ними по ключевому полю VBELN.  
Стоимость позиции находится в поле NETWR на уровне позиций заказов, номер клиента в поле KUNNR.  
Дату документа проверять по полю VBAK-AUDAT.  
Выборку из двух таблиц осуществить одним селектом через JOIN. 
 
Выбранные и агрегированные данные вывести в ALV со следующими полями: 
ГОД 
VBAK-KUNNR 
VBAP-MATNR 
VBAP-ARKTX 
VBAP-KWMENG (количество, агрегированное) 
VBAP-MEINS (ед измерения) 
VBAP-NETWR (стоимость позиции, агрегированное) 
 
Агрегированние данных реализовать двумя способами:  
1) с помощью суммирования во внутренней таблице (или селекте)  
2) и с помощью функций ALV, применяемых к исходному набору данных. 
 
На селекционном экране разместить поля для ограничения выборки: 
Год – обязательное поле типа GJAHR, дефолтное значение 2021 
Номер клиента – тип VBAP-KUNNR, необязательное поле 
Номер материала – тип VBAP-MATNR, необязательное поле 
Переключатель radiobutton, который определяет способ агрегации: через программирование или через функции ALV. 
 
К программе привязать транзакцию.
